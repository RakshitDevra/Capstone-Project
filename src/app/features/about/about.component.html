About This Application

The Product Inventory Management System is a Single Page Application (SPA) designed to simplify and streamline inventory management for businesses.

This platform enables users to:

  • View, add, update, and delete products
  • Organize products by categories
  • Update stock quantities
  • Manage product details including descriptions, prices, and manufacturers
  • Perform real-time inventory tracking
  • Allow administrators to securely log in and manage inventory

Built using Angular for the front-end and a JSON server for the back-end, this system ensures smooth and efficient product management.
